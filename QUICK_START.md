# 快速开始 - 增强版 AI 导演工作台

## 5 分钟上手指南

---

## 第一步：启动应用

```bash
# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

访问：http://localhost:5173

---

## 第二步：进入增强版工作台

**方式 1**：从首页进入
1. 点击顶部"更新通知"标签
2. 找到"🎉 增强版AI导演工作台"
3. 点击进入

**方式 2**：直接访问
- 访问：http://localhost:5173/enhanced-director

---

## 第三步：上传参考素材（可选但推荐）

### 角色参考图
1. 点击右侧"参考素材管理"
2. 选择"👤 角色参考"标签
3. 点击"+"上传角色图片
4. 系统自动分配 @图片1、@图片2...

**建议**：
- 每个主要角色准备 1-2 张参考图
- 保持光线、角度、服装一致
- 图片清晰，分辨率 512x512 以上

### 场景参考图
1. 选择"🏛️ 场景参考"标签
2. 上传场景图片
3. 系统自动分配编号

**建议**：
- 每个主要场景准备 1 张参考图
- 可以是实景照片或 AI 生成图
- 风格统一

---

## 第四步：选择生成模式

### 普通模式（推荐新手）
- 适合：标准分镜制作
- 特点：逐个生成，自动连贯
- 用途：大部分场景

### 一镜到底模式
- 适合：需要连贯性的长镜头
- 特点：无切镜，沉浸感强
- 用途：展现空间关系、时间连续性

### 音乐卡点模式
- 适合：MV、广告片
- 特点：节奏精准同步
- 用途：节奏感强的内容

---

## 第五步：生成提示词

1. 点击"生成 Seedance 2.0 提示词"按钮
2. 等待 1-2 秒
3. 查看生成结果

**生成内容包括**：
- ✅ 中文提示词（主要内容）
- ✅ 运镜关键词
- ✅ 画幅比例
- ✅ 时长
- ✅ 情绪强度
- ✅ 参考图引用
- ✅ 连贯性说明

---

## 第六步：质量检查

系统自动进行质量检查，显示：

### ✅ 通过
```
提示词质量良好，可以直接使用
```

### ⚠️ 警告

## 第七步：使用提示词

### 方式 1：复制到 Seedance 2.0

1. 复制生成的中文提示词
2. 打开 Seedance 2.0 平台
3. 粘贴提示词
4. 上传对应的参考图（如果有）
5. 设置画幅、时长等参数
6. 点击生成

### 方式 2：保存为文件

1. 复制提示词
2. 保存到本地文件
3. 后续批量使用

## 常见问题

### Q1：必须上传参考图吗？

**A**：不是必须的，但强烈建议。参考图可以大幅提升角色和场景的一致性（+80%）。

### Q2：生成的提示词可以修改吗？

**A**：可以。生成后可以手动编辑提示词，但建议保持中文和运镜关键词。

### Q3：一镜到底模式适合多长的镜头？

**A**：建议 10-15 秒。太长可能影响质量。

### Q4：音乐卡点模式需要什么？

**A**：需要提供音乐的 BPM（节拍）和关键卡点时间点。

### Q5：生成失败怎么办？

**A**：查看质量检查的警告和建议，根据提示优化提示词。

## 进阶技巧

### 1. 批量生成

选择"普通模式"，系统会自动为所有镜头生成提示词，并处理连贯性。

### 2. 自定义运镜

在生成后手动修改运镜关键词，如：
- 固定镜头 → 推镜头
- 中景 → 特写

### 3. 情绪强化

在提示词中添加更多情绪词汇：
- 紧张 → 紧张激烈、强烈的对抗感
- 温柔 → 柔和温暖、细腻的情感

### 4. 参考图组合

为同一角色准备多个参考图：
- @图片1：正面照
- @图片2：侧面照
- @图片3：背面照

根据镜头需要选择合适的参考图。
