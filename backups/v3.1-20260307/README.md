## 备份时间

## 版本说明
此版本在v3.0基础上增加了以下功能：

### 新增功能
1. **工作台通知模块**
   - 更新通知Tab（生成队列+系统通知+平台更新）
   - 通知筛选功能
   - 平台更新与帮助文档

2. **剧本工坊优化**
   - AI自定义修改工具（剧本编辑Tab内侧边面板）
   - 开头精修、续写剧本等固化工具
   - 快速模板

3. **选角中心优化**
   - 角色三视图+特写展示
   - 音色配音下拉选择

4. **分镜导演台优化**
   - 旁白音色配置
   - AI优化提示词面板（支持直接优化/自定义修改）

5. **成片输出优化**
   - 移除比例设置
   - 新增输出到交片中心、输出到翻译中台
   - 整部剧操作说明
