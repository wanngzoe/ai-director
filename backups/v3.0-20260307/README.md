## 版本说明
此版本包含以下功能：

### 页面 (8个)
1. 工作台 - 项目列表视图
2. 剧本工坊 - 剧本导入与解析
3. 选角中心 - 角色管理
4. 场景工坊 - 场景管理
5. 道具管理 - 道具管理
6. 分镜导演台 - **核心页面（左右两栏布局）**
7. 生成队列 - 任务监控
8. 后期精修 - 时长调节与字幕
9. 成片输出 - 导出

### 主要优化
1. 分镜导演台左右两栏布局（视频左，配置右）
2. 9:16竖屏视频预览（320px宽）
3. 视频生成模式切换（Seedance 2.0 / Wan 2.6）
4. 连续播放预览按钮
5. 视频操作按钮：生成视频、字幕擦除、视频裁剪、变速调节
6. 进度条合并"分镜&视频生成"
7. 去除重复的生成中状态
