# 生成队列产品功能概述

## 产品定位

生成队列是 AI 视频生成任务的管理中心，帮助用户可视化追踪和管理所有视频生成任务的状态。

---

## 核心功能

### 1. 任务展示
以列表形式展示所有生成任务，每行包含：
- **项目名称**：所属项目
- **剧集**：第X集
- **镜头**：第X个镜头
- **状态**：等待中 / 生成中 / 已完成 / 失败
- **进度**：生成进度百分比 或 排队信息
- **创建时间**：任务提交时间
- **完成时间**：任务完成时间
- **操作**：查看结果 / 重试

### 2. 状态管理
| 状态 | 说明 | 显示 |
|------|------|------|
| 等待中 | 任务排队等待 | 显示排队位置 + 预计等待时间 |
| 生成中 | 正在生成视频 | 进度条 + 百分比 |
| 已完成 | 生成成功 | 绿色勾 + 完成时间 |
| 失败 | 生成失败 | 红色感叹号 + 重试按钮 |

### 3. 排队信息（针对等待中任务）
- **排队位置**：当前排在第几位 / 总排队数
- **预计等待时间**：根据排队位置计算（每任务约2.5分钟）

### 4. 统计概览
顶部显示三类任务数量：
- 进行中
- 等待中
- 已完成

---

## 用户交互流程

1. 用户在分镜导演模块提交生成任务
2. 任务进入生成队列
3. 用户可在队列页面查看所有任务状态
4. 完成后可点击"查看"预览结果
5. 失败任务可点击"重试"
