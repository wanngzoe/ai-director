# 分镜导演台产品功能概述

## 产品定位

分镜导演台是 AI 视频生成的核心工作区，支持从分镜设计到视频生成的全流程管理。

---

## 核心功能

### 1. 集数与镜头管理
- **集数选择**：支持多集管理，可切换不同集数
- **镜头序列**：以时间线形式展示所有镜头
- **镜头状态**：待生成 / 生成中 / 已完成
- **快速预览**：点击镜头卡片查看详情

### 2. 视频预览区
- **预览窗口**：9:16 竖屏比例展示
- **状态覆盖层**：
  - 生成中显示加载动画和预计剩余时间
  - 已完成显示视频缩略图
- **快捷操作**：
  - 生成视频
  - 字幕擦除
  - 视频裁剪
  - 变速调节

### 3. 生成设置

#### 视频生成模型选择
| 模型 | 特点 |
|------|------|
| Seedance 2.0 | 全能参考模式，支持多图/视频参考 |
| Wan 2.6 | 首帧图生成模式 |

#### 旁白音色
- 多种 AI 音色可选（温柔女声、甜美女声、成熟男声等）
- 旁白文字自定义输入

#### 提示词设置
- **提示词模式**：自然模式 / 详细模式 / 简洁模式
- **镜头描述**：包含场景、角色、动作描述
- **运镜控制**：推、拉、摇、移、环绕等
- **氛围设定**：光线、色调、情绪

### 4. 参考图管理
- **角色参考图**：关联角色形象
- **场景参考图**：关联场景环境
- **上传方式**：支持多图上传

### 5. 连续性控制
- **上一镜头参考**：保持角色/场景一致性
- **角色一致性权重**：调整连续性强度

### 6. 批量操作
- **连续播放预览**：预览多个镜头
- **批量生成**：一键生成多个镜头

---

## 用户交互流程

1. **选择集数**：从下拉菜单选择要编辑的集数
2. **选择镜头**：点击镜头卡片，右侧显示详情
3. **配置生成参数**：
   - 选择视频生成模型
   - 设置旁白音色
   - 调整提示词
   - 上传参考图
4. **生成视频**：点击生成按钮
5. **查看结果**：生成完成后可预览和后期处理
