## 技术参数推荐

### 画幅比例
- 16:9 - 横屏（建立、全景、越肩）
- 9:16 - 竖屏（中景、特写）
- 1:1 - 方形（大特写、创意）

### 时长建议
- 建立镜头：3-5秒
- 对话镜头：4-6秒
- 特写镜头：2-4秒
- 动作镜头：4-8秒
- 转场镜头：2-3秒

### 运动强度
- 静态场景：0.2-0.3
- 轻微运动：0.4-0.5
- 中等运动：0.6-0.7
- 强烈运动：0.8-0.9

## 分镜设计黄金法则

1. **180度轴线规则** - 保持拍摄方向一致
2. **30度规则** - 相邻镜头角度差至少30度
3. **视线匹配** - 角色视线方向要连贯
4. **动作匹配** - 动作要连续流畅
5. **情绪递进** - 镜头大小随情绪变化
6. **节奏控制** - 时长影响节奏感

## 常见错误避免

❌ 不要：
- 镜头跳跃太大（远景直接到大特写）
- 相机运动过于频繁
- 提示词过长（超过200词）
- 忽略照明描述
- 缺少情绪氛围

✅ 应该：
- 镜头大小渐进变化
- 静态为主，运动为辅
- 提示词简洁精准
- 明确照明风格
- 强调情绪氛围

## 快速决策流程图

```
场景类型？
├─ 对话 → 正反打（中景+越肩）
├─ 动作 → 跟随镜头（全景+中景）
├─ 情绪 → 特写（特写+大特写）
└─ 建立 → 远景（静态+广角）

情绪强度？
├─ 高 → 特写/大特写
├─ 中 → 中景/特写
└─ 低 → 全景/中景

相机运动？
├─ 对话 → 静止
├─ 动作 → 跟随/手持
├─ 情绪 → 推进/静止
└─ 建立 → 静止/摇移
```

## 一分钟速成指南

1. **看剧本** → 识别场景类型（对话/动作/情绪）
2. **定镜头** → 选择镜头类型（远/全/中/特）
3. **设运动** → 确定相机运动（静/跟/推）
4. **配照明** → 选择照明风格（自然/低调/高调）
5. **写提示词** → 组合关键词（类型+动作+照明+情绪）
6. **选模型** → 根据需求优化（Seedance/Wan/Runway）
