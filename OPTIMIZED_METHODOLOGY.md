# 优化后的 AI 导演方法论

> 基于 Seedance 2.0 多模态能力的完整工作流

---

## 核心优化点

### 1. 提示词语言：必须使用中文 ✅

**原方案**：使用英文提示词
```
wide establishing shot, luxurious CEO office, night time, static camera...
```

**优化后**：使用自然中文
```
总裁办公室，夜晚灯光氛围，远景，展现豪华的办公环境，
落地窗外城市夜景，固定镜头，紧张的氛围
```

**原因**：Seedance 2.0 对中文理解更准确，响应更精准

---

### 2. 运镜关键词：使用中文标准词汇 ✅

**必须使用的中文运镜关键词**：

| 类型 | 关键词 | 说明 |
|------|--------|------|
| 固定 | 固定镜头、静止镜头 | 相机不动 |
| 推拉 | 推镜头、拉镜头、推近、拉远 | 前后移动 |
| 摇动 | 摇镜头、左右摇镜、上下摇镜 | 旋转 |
| 移动 | 移镜头、横向移动、纵向移动 | 平移 |
| 跟随 | 跟随镜头、跟拍 | 跟随主体 |
| 环绕 | 环绕镜头、360度环绕 | 围绕拍摄 |
| 角度 | 俯拍、仰拍、低角度、高角度 | 拍摄角度 |
| 景别 | 特写、中景、远景、全景 | 画面范围 |
| 特殊 | 一镜到底、切镜、切换镜头 | 特殊技法 |

**示例**：
```
推镜头，从远景推近到人物特写
跟随镜头，跟拍角色在走廊中奔跑
环绕镜头，360度展现办公室环境
```

---

### 3. 参考图/视频引用：@图片X @视频X ✅

**新增功能**：支持多模态输入

**格式**：
- `@图片1`、`@图片2` = 参考图片
- `@视频1`、`@视频2` = 参考视频

**使用场景**：

#### 场景1：角色一致性
```
林婉儿的形象参考@图片1，穿着职业装，
陆霆琛的形象参考@图片2，西装笔挺
```

#### 场景2：场景参考
```
总裁办公室场景参考@图片3，
豪华现代的室内设计，落地窗外城市夜景
```

#### 场景3：运镜复刻
```
完全参考@视频1的运镜效果和节奏，
推镜头从远景到特写，流畅自然
```

#### 场景4：动作参考
```
角色的动作参考@视频1中的打斗场面，
动作流畅有力，充满张力
```

---

### 4. 一镜到底：连贯性大幅提升 ✅

**新能力**：Seedance 2.0 支持长镜头连贯拍摄

**使用方法**：
```
一镜到底，总裁办公室，夜晚，
镜头从门口推进，展现办公室全景，
然后横向移动，跟随林婉儿走到窗前，
接着推近到她的面部特写，捕捉紧张的表情，
最后拉远，陆霆琛从背后进入画面
```

**适用场景**：
- 需要展现空间关系
- 强调时间连续性
- 营造沉浸感
- 复杂的场景调度

**技术要点**：
- 明确说明"一镜到底"
- 详细描述每个运镜转换
- 保持动作的连贯性
- 总时长建议10-15秒

---

### 5. 音乐卡点：节奏精准控制 ✅

**新能力**：根据音乐节奏自动卡点

**使用方法**：
```
根据@视频1的音乐节奏进行卡点，
0-3秒：林婉儿转身，特写面部，
3-5秒：切镜，陆霆琛推门而入，
5-8秒：双人中景，对视，
8-10秒：快速切换，情绪高潮，
画面切换与音乐节奏完美同步
```

**适用场景**：
- 音乐视频(MV)
- 广告片
- 宣传片
- 节奏感强的内容

---

### 6. 视频延长：无缝扩展 ✅

**新能力**：延长现有视频，保持风格一致

**使用方法**：
```
将@视频1延长10秒，
林婉儿继续站在窗前，
镜头缓缓推近，
她转身看向门口，
表情从紧张变为坚定，
保持原视频的风格和节奏
```

**适用场景**：
- 视频时长不够
- 需要补充过渡
- 延展情绪
- 增加铺垫

---

### 7. 视频编辑：精准修改 ✅

**新能力**：修改视频的特定部分

**使用方法**：
```
修改@视频1的3-6秒部分，
将林婉儿的表情从紧张改为愤怒，
她猛地转身，眼神犀利，
保持其他部分不变
```

**适用场景**：
- 修正表情
- 调整动作
- 改变情绪
- 替换元素

---

### 8. 情绪演绎：更加细腻 ✅

**优化**：情绪描述更具体、更视觉化

**原方案**：
```
emotion: "angry"
```

**优化后**：
```
紧张激烈的氛围，强烈的对抗感，
角色面部表情愤怒，眼神犀利，
肢体语言充满攻击性，
快速的呼吸，紧握的拳头
```

**情绪强度分级**：

| 强度 | 情绪类型 | 视觉特征 | 镜头建议 |
|------|----------|----------|----------|
| 高 | 愤怒、恐惧、崩溃 | 强对比、快节奏 | 特写、手持 |
| 中 | 高兴、悲伤、惊讶 | 中等对比 | 中景、特写 |
| 低 | 平静、思考、观察 | 柔和、稳定 | 中景、远景 |

### 9. 连贯性处理：自动衔接 ✅

**新增**：自动生成镜头间的连贯性说明

**示例**：
```
镜头1：林婉儿站在窗前，背对门口
镜头2：陆霆琛推门而入
连贯性说明：保持林婉儿的位置和姿态，
确保陆霆琛进门的方向与镜头1的空间关系一致
```

**连贯性要素**：
- 角色位置关系
- 动作衔接
- 情绪延续
- 光线一致性
- 空间逻辑

## 完整工作流（优化版）

### 阶段1：剧本解析（不变）

输入剧本 → 提取场景信息 → 结构化数据

### 阶段2：分镜设计（增强）

**新增考虑因素**：
1. 是否需要一镜到底？
2. 是否有音乐卡点需求？
3. 是否需要参考视频运镜？
4. 镜头间的连贯性如何处理？

### 阶段3：提示词生成（全面优化）

**生成流程**：
```typescript
1. 基础提示词生成（中文）
   ↓
2. 添加参考图/视频引用
   ↓
3. 优化运镜关键词
   ↓
4. 添加连贯性说明
   ↓
5. 质量检查和验证
   ↓
6. 输出最终提示词
```

## 提示词质量检查清单

### 必须包含 ✅
- [ ] 使用中文描述
- [ ] 包含运镜关键词
- [ ] 场景描述具体
- [ ] 角色描述清晰
- [ ] 动作描述视觉化
- [ ] 情绪氛围明确

### 建议包含 ⭐
- [ ] 参考图/视频引用
- [ ] 连贯性说明
- [ ] 光线描述
- [ ] 构图说明
- [ ] 对话内容

### 避免 ❌
- [ ] 使用英文提示词
- [ ] 提示词过长（>500字）
- [ ] 提示词过短（<20字）
- [ ] 缺少运镜描述
- [ ] 动作描述模糊
- [ ] 忽略连贯性
