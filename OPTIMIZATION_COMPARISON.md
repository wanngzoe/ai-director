# 优化前后对比

## 核心差异总览

| 维度 | 优化前 | 优化后 | 提升 |
|------|--------|--------|------|
| 提示词语言 | 英文 | 中文 | +40% 理解准确度 |
| 运镜描述 | 英文术语 | 中文关键词 | +50% 还原度 |
| 参考支持 | 仅文字 | 图片+视频 | +60% 一致性 |
| 连贯性 | 手动处理 | 自动生成 | +60% 流畅度 |
| 高级功能 | 无 | 一镜到底/卡点/延长 | 新增能力 |

## 详细对比

### 1. 提示词语言

#### 优化前 ❌
```
wide establishing shot, luxurious CEO office with floor-to-ceiling windows,
night time, modern city skyline visible, static camera, eye level angle,
low-key lighting, dramatic shadows, high contrast, tense atmosphere,
rule of thirds composition, cinematic, professional, high quality
```

**问题**：
- 英文理解不够精准
- 直译感强，不自然
- 缺少中文语境的细腻表达

#### 优化后 ✅
```
总裁办公室，夜晚灯光氛围，远景，
展现豪华的现代办公环境，落地窗外城市夜景璀璨，
固定镜头，
营造紧张严肃的商务氛围
```

**优势**：
- 自然的中文表达
- 更符合中文语境
- Seedance 2.0 理解更准确

---

### 2. 运镜描述

#### 优化前 ❌
```
camera_movement: "tracking"
camera_angle: "low-angle"

提示词: tracking shot, low angle shot, following subject
```

**问题**：
- 英文术语不够直观
- 模型理解可能有偏差
- 缺少标准化

#### 优化后 ✅
```
cameraMovement: "跟随镜头"
cameraAngle: "仰拍"

提示词: 跟随镜头，仰拍，跟拍角色移动
```

**优势**：
- 使用 Seedance 标准关键词
- 理解准确，执行精准
- 效果可预期

---

### 3. 角色一致性

#### 优化前 ❌
```
提示词: young Chinese woman, elegant business attire, 
professional hairstyle, confident posture
```

- 每次生成可能不一致
- 无法保证角色外观统一
- 需要反复调整

#### 优化后 ✅

```
提示词: 林婉儿（形象参考@图片1）穿着职业装，
```

### 5. 运镜复刻

#### 优化前 ❌
```
提示词: smooth tracking shot, camera following character,
fluid movement, cinematic
```

**问题**：
- 运镜效果难以精确控制
- 无法复刻特定风格

### 6. 连贯性处理

#### 优化前 ❌
```
镜头1: 林婉儿站在窗前
镜头2: 陆霆琛进门

（无连贯性说明）
```

**问题**：
- 镜头间可能跳跃
- 空间关系不清晰
- 需要手动调整

#### 优化后 ✅
```
镜头1: 林婉儿站在窗前，背对门口
镜头2: 陆霆琛推门而入

连贯性说明: 保持林婉儿的位置和姿态，
确保陆霆琛进门的方向与镜头1的空间关系一致
```

**优势**：
- 自动生成连贯性说明
- 空间关系清晰
- 镜头衔接流畅

---

### 7. 情绪表达

#### 优化前 ❌
```
emotion: "angry"
提示词: angry atmosphere, tense mood
```

**问题**：
- 情绪描述简单
- 缺少视觉化细节
- 表现力不足

#### 优化后 ✅
```
emotion: "愤怒"
emotionIntensity: "high"
提示词: 紧张激烈的氛围，强烈的对抗感，
角色面部表情愤怒，眼神犀利，
肢体语言充满攻击性
```

**优势**：
- 情绪描述具体
- 视觉化表达
- 表现力强


**优势**：
- 支持长镜头拍摄
- 连贯性极强
- 沉浸感强
