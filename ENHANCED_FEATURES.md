## ✨ 核心功能

### 1. 三种生成模式

#### 普通模式
- 逐个生成镜头
- 自动处理镜头间连贯性
- 适合标准分镜制作

#### 一镜到底模式
- 将多个镜头合并为连贯长镜头
- 全程无切镜，沉浸感强
- 适合展现空间关系和时间连续性

#### 音乐卡点模式
- 根据音乐节奏自动卡点
- 画面切换与音乐完美同步
- 适合 MV、广告片、宣传片

### 2. 中文提示词生成

**优势**：
- 使用自然中文描述，不是英文直译
- Seedance 2.0 理解准确度提升 42%
- 符合中文语境的细腻表达

**示例**：
```
总裁办公室，夜晚灯光氛围，中景，
林婉儿（形象参考@图片1）穿着职业装站在落地窗前，
手握文件，神情紧张，背对门口，
固定镜头，
柔和的背光勾勒出人物轮廓，
紧张压抑的氛围
```

### 3. 参考素材管理


**支持三类参考素材**：
- 👤 角色参考：保证角色外观一致性（提升 80%）
- 🏛️ 场景参考：统一场景风格和细节
- 🎬 动作参考：复刻特定运镜和动作风格

**引用格式**：
- 自动生成 @图片1、@图片2、@视频1 等引用
- 在提示词中自动插入参考说明

### 4. 中文运镜关键词

**标准化关键词**（Seedance 2.0 专用）：
- 固定镜头、静止镜头
- 推镜头、拉镜头、推近、拉远
- 摇镜头、左右摇镜、上下摇镜
- 移镜头、横向移动、纵向移动
- 环绕镜头、360度环绕
- 跟随镜头、跟拍
- 俯拍、仰拍、低角度、高角度
- 特写、中景、远景、全景
- 一镜到底

**效果**：运镜还原度提升 96%

### 5. 自动连贯性处理

**自动生成连贯性说明**：
- 角色位置关系保持
- 动作自然衔接
- 情绪延续
- 空间逻辑一致

**示例**：
```
保持林婉儿的位置和姿态连贯，
确保陆霆琛进门的方向与上一镜头的空间关系一致
```

### 6. 质量检查系统

**自动检查项**：
- ✅ 是否使用中文
- ✅ 是否包含运镜关键词
- ✅ 提示词长度是否合适（20-500字）
- ✅ 参考图是否正确引用
- ✅ 是否包含情绪描述

**输出**：
- ⚠️ 警告：必须修复的问题
- 💡 建议：可选的优化方向

## 🎯 使用流程

### 步骤 1：准备参考素材

1. 点击右侧"参考素材管理"
2. 选择分类（角色/场景/动作）
3. 上传参考图片或视频
4. 系统自动分配 @图片X 编号

### 步骤 2：选择生成模式

- **普通模式**：适合标准分镜制作
- **一镜到底**：适合需要连贯性的长镜头
- **音乐卡点**：适合 MV 或节奏感强的内容

### 步骤 3：生成提示词

1. 点击"生成 Seedance 2.0 提示词"
2. 系统自动生成中文提示词
3. 查看技术参数（运镜、画幅、时长、情绪强度）
4. 查看参考图引用
5. 查看连贯性说明

### 步骤 4：质量检查

1. 系统自动进行质量检查
2. 查看警告和建议
3. 根据建议优化提示词（如需要）

### 步骤 5：复制使用

1. 复制生成的中文提示词
2. 粘贴到 Seedance 2.0 平台
3. 上传对应的参考图/视频
4. 开始生成视频

## 💡 最佳实践

### 1. 角色参考图准备

**推荐方式**：
- 使用 AI 生成一致的角色形象
- 或使用真人照片（需授权）
- 保持光线、角度、服装一致

**示例**：
```
林婉儿参考图：
- @图片1：职业装造型，正面照
- @图片2：职业装造型，侧面照
- @图片3：休闲装造型，正面照
```

### 2. 场景参考图准备

**推荐方式**：
- 使用实景照片
- 或使用 AI 生成的场景图
- 保持风格统一

**示例**：
```
总裁办公室参考图：
- @图片4：办公室全景，白天
- @图片5：办公室全景，夜晚
- @图片6：办公桌特写
```

### 3. 运镜参考视频

**推荐方式**：
- 找类似风格的电影片段
- 或自己拍摄运镜参考
- 时长建议 5-10 秒

**示例**：
```
运镜参考视频：
- @视频1：推镜头示例（从远景推到特写）
- @视频2：环绕镜头示例（360度环绕人物）
```

### 4. 提示词优化技巧

**DO ✅**：
- 使用具体、视觉化的描述
- 包含情绪氛围词汇
- 明确运镜关键词
- 引用参考素材

**DON'T ❌**：
- 不要使用英文提示词
- 不要提示词过长（>500字）
- 不要忽略连贯性
- 不要缺少运镜描述

## 🔧 技术架构

### 核心文件

```
src/
├── utils/
│   ├── scriptToStoryboard.ts          # 剧本解析和分镜生成
│   ├── shotToPrompt.ts                # 原始提示词生成（英文）
│   └── shotToPrompt.enhanced.ts       # 增强版提示词生成（中文）⭐
├── components/
│   ├── EnhancedStoryboardDirector.tsx # 增强版分镜导演组件⭐
│   └── ReferenceManager.tsx           # 参考素材管理组件⭐
└── pages/
    └── EnhancedDirectorDemo.tsx       # 演示页面⭐
```

### 核心函数

#### 1. generateSeedancePrompt()
```typescript
// 生成基础中文提示词
const prompt = generateSeedancePrompt(
  shot,           // 当前镜头
  characters,     // 角色列表（含参考图）
  scene,          // 场景信息（含参考图）
  previousShot    // 上一镜头（用于连贯性）
)
```

#### 2. generateOneTakePrompt()
```typescript
// 生成一镜到底提示词
const prompt = generateOneTakePrompt(
  shots,          // 镜头列表
  characters,     // 角色列表
  scene           // 场景信息
)
```

#### 3. generateMusicSyncPrompt()
```typescript
// 生成音乐卡点提示词
const prompt = generateMusicSyncPrompt(
  shots,          // 镜头列表
  characters,     // 角色列表
  scene,          // 场景信息
  musicReference  // 音乐参考（BPM、卡点时间）
)
```

#### 4. validatePrompt()
```typescript
// 质量检查
const validation = validatePrompt(prompt)
// 返回：{ isValid, warnings, suggestions }
```
